# 食品日期管理 - 安全小管家

## 專案簡介
「食品日期管理 - 安全小管家」是一個專為家庭設計的食品保質期管理工具，幫助用戶輕鬆記錄、追蹤和管理冰箱中的食物，減少食物浪費，守護家人健康。

## 主要功能

### 1. 智能保質期追蹤
- **自動計算剩餘天數**：根據購買日期和有效期限自動計算食物剩餘保存時間
- **視覺化狀態顯示**：使用紅綠燈系統（紅：過期、黃：注意、綠：新鮮）直觀顯示食物狀態
- **批量管理**：支援多種食物同時記錄與管理

### 2. 自動標籤分類
- **智能分類**：系統自動辨識食物類型並分類（乳製品、肉類、蔬菜、飲料等）
- **自定義標籤**：用戶可根據需求創建個性化標籤
- **快速查找**：按分類快速篩選和查找食物

### 3. 到期提醒通知
- **多管道提醒**：支援網頁推播通知
- **自定義提醒時間**：可設定過期前3天、1天等不同時間點的提醒
- **即時狀態更新**：系統自動掃描並更新食物狀態

### 4. 趣味知識測驗
- **食物保存知識測試**：透過互動測驗學習正確的食物保存方法
- **即時反饋**：答題後立即顯示正確答案與解釋
- **學習激勵**：完成測驗可獲得成就鼓勵

### 5. 問題回報與反饋
- **多類型回報**：支援系統錯誤、功能建議、資料問題等回報類型
- **圖文並茂**：可上傳截圖輔助問題描述
- **即時確認**：提交後顯示成功確認訊息

## 技術特色

### 前端技術
- **響應式設計**：使用Tailwind CSS實現全設備兼容
- **現代交互**：Vanilla JavaScript實現平滑動畫和互動功能
- **無障礙訪問**：符合WCAG標準，支援螢幕閱讀器
- **性能優化**：懶加載、平滑滾動、減少動畫等優化

### 使用者體驗
- **直觀界面**：簡潔美觀的設計，降低學習成本
- **流暢導航**：錨點平滑滾動，清晰的頁面結構
- **移動優先**：專為手機和平板優化的操作體驗
- **即時反饋**：所有操作都有視覺或文字反饋

## 使用流程

1. **查看食物清單**：在首頁查看已記錄的食物狀態
2. **輸入食物資料**：輸入食物名稱、購買日期和保存期限
3. **智能分類排序**：系統自動分類並按保質期排序
4. **接收提醒**：在食物即將過期時收到通知
5. **定期檢查**：定期查看食物狀態，及時處理即將過期的食物
